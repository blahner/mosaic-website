Frequently Asked Questions

What file formats are supported?

Currently, we only support HDF5 files (.hdf5 or .h5 extensions). These files should contain the required metadata attributes for proper organization and discovery.

How are files organized?

Files are grouped by dataset name, pipeline, trial format, publication URL, and GitHub repository URL. Files sharing these exact attributes will be grouped together on the download page.

How do I download files?

On the download page, select the files you want by checking the boxes. A bash script will be automatically generated that you can copy and run on your local machine to download the selected files via AWS CLI.

What do I need to run the download script?

You need to have the AWS CLI installed and configured with appropriate credentials to access the S3 bucket. The script uses standard AWS CLI commands to download files.

What metadata is required?

Required metadata includes: unique file ID, dataset name, subject name, pipeline, owner name, owner email, and trial format. Optional metadata includes GitHub repository URL and publication URL.

Can I search for specific datasets?

Yes, use the search box on the download page to filter datasets by dataset name, subject name, pipeline, owner name, or file ID.

How do I upload files?

Use the upload page to select your HDF5 file and fill in the required metadata. The file will be uploaded to S3 with the metadata embedded or associated with it.

Is there a file size limit?

File size limits depend on your S3 bucket configuration and AWS account limits. For very large files, consider using multipart upload or contact your administrator.
